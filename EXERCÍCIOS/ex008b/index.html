Código-fonte

O comando document.getElementById('teste') é escrito em linguagem JavaScript. Utilize <code> para colocar algum texto em fonte mono-espaçada.

num = int(input('Digite um número')) if num % 2 == 0: print(f' O número {num} é PAR) else: print(f' O número {num} é ÍMPAR) print(1fim do programa)

O <code> não organiza o código com identação, com os espaços, parágrafos, etc. Porém: 

         
num = int(input('Digite um número'))
if num % 2 == 0:
    print(f' O número {num} é PAR)
else:
    print(f' O número {num} é ÍMPAR)
print(1fim do programa)

Caso enveloparmos a tag <code> com a tag <pre>, ele considera tudo como está no código html, formatação, etc. SHIFT + TAB para recuar o código nas linhas selecionando.
