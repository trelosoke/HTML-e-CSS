Links: definição e atributos

Acesse meu site aqui.

A tag <a>; tem a função de ancorar uma palavra com um link clicável que redirecione para algum URL. Existem 3 atributos que podem ser usados na tag: "href" (obrigatório); "target" e "rel".

Atributo href

O href é utilizado para colocar o caminho do link, o URL.

Atributo target

Utilizado com 2 valores iniciais: _self e _blank. O _blank significando que, ao se redirecionar para um link, outra guia do navegador seria aberta e _self significando sobrepor o site atual pelo site do link clicado. Geralmente _self é usado para links do próprio sites (outras páginas do mesmo domínio). E _blank para redirecionar sites externos, sem ser do mesmo domínio. Os outros valores de target envolvem coisas mais complexas.

Atributo rel

Significa o relacionamento (relationship) que o link tem com o target. É mais um atributo que auxilia o buscador a identificar o tipo de link que está usando, ou até fazer com que ele não recomende o link (nofollow), entre outros valores. Como anteriormente foi dito, a HTML5 éuma liguagem muito mais semântica e ao longo das atualizações, será cada vez mais assim.

Lista de valores de rel:

  • noopener : Impede que a nova página tenha acesso ao window.opener, melhorando a segurança;
  • noreferrer : Não envia informações de referência da página ao destino;
  • nofollow : Indica aos mecanismos de busca para não seguir esse link;
  • ugc : Define que o link foi gerado por um usuário, útil para comentários e fóruns;
  • author : Especifica que o link leva ao autor do documento atual;
  • alternate : Indica uma versão alternativa do conteúdo, como um feed RSS;
  • bookmark : Marca o link como um favorito ou referência;
  • external
  • help
  • license
  • prev/next
